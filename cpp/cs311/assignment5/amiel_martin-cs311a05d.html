Exercise A — Mathematical Induction

Answer each of the following questions.

  1. Basic Ideas.
    1. Name and briefly explain the three parts of an induction proof, as covered in class.
      1. Basis: prove that the statement is true for some small n
      2. Inductive hyponthesis: assume that the statement is true for n + 1
      3. Inductive conclusion: prove that the statement is true for n + 1
    2. What is the difference between “weak induction” and “strong induction”?
      • in weak induction, we assume that the statement is true for all numbers of n
      • in strong induction, we assume that the statement is true for all number of n and all smaller values of n that are valid

     
  2. Mathematical Induction & Functions.
    1. Mathematical induction is particularly useful for verifying facts about a certain kind of function. What kind is this?
      2. recursive functions OR functions that implement recursive algorithms
    2. We discussed two things related to a function that could be verified using a proof. Name these two things.
      1. correctness
      2. efficiency

     
  3. Pieces of a Proof. Suppose we are proving that f(n) = g(n), for every positive integer n, using mathematical induction.
    1. What must we prove, in the basis (base case)?
      2. we must prove that f(1) == g(1)
    2. If we are using weak induction, what is our induction hypothesis?
      3. f(n + 1) == g(n + 1)

  2. How fast are the slowest algorithms that we would generally consider to be useful?
    O(n log n)
